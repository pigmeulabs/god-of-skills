# Pigmeu Never Forget — Brand assets

- Vector assets (SVG) for mascot logo and icon
- Visual language mirrors the Pigmeu Copilot reference: pygmy rider, angular dark beast, cyan glowing eye, and bold/tall wordmark
- Logo variants keep transparent background for flexible use in docs, README, site headers, and social cards

Files:
- `pigmeu-never-forget-logo-horizontal.svg`: main horizontal lockup
- `pigmeu-never-forget-logo.svg`: stacked logo for covers and centered layouts
- `pigmeu-never-forget-icon.svg`: mascot-only icon for avatar, app icon, and favicon derivations

Usage notes:
- Prefer the horizontal logo for headers and repository pages.
- Use the stacked logo where vertical balance matters.
- Use the square icon when text would become unreadable at small sizes.
- Keep transparent background; when placing on dark surfaces, verify contrast around the wordmark.

Directory: `assets/brand/`
