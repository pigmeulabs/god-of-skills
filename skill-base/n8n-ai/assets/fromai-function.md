# $fromAI() Function Reference

Complete reference for the $fromAI() function in n8n.

---

## What is $fromAI()?

Allows the AI model to dynamically fill in parameters of tools connected to the AI Agent.

---

## Syntax

```javascript
{{ $fromAI('key', 'description', 'type', defaultValue) }}
```

---

## Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `key` | string | Yes | Argument name (1-64 chars, only letters, numbers, underscores, hyphens) |
| `description` | string | No | Description of the argument for the AI |
| `type` | string | No | `string`, `number`, `boolean`, or `json` (default: `string`) |
| `defaultValue` | any | No | Default value if AI can't find context |

---

## Examples

### Simple
```javascript
{{ $fromAI('email') }}
```

### Complete
```javascript
{{ $fromAI('name', "The commenter's name", 'string', 'Jane Doe') }}
```

### Numeric
```javascript
{{ $fromAI('numItemsInStock', 'Number of items in stock', 'number', 5) }}
```

### Partial (fixed prefix + dynamic value)
```javascript
Generated by AI: {{ $fromAI('subject') }}
```

---

## Important Notes

- Only available for tools connected to the AI Agent
- Does NOT work with Code tool or other non-tool sub-nodes
- Arguments are "hints" for the AI, not references to existing values
- In chat workflows, the AI may ask the user for information if not found in context

---

## Use Cases

### Dynamic API Parameters
```javascript
// HTTP Request Tool URL
https://api.example.com/users/{{ $fromAI('userId', 'The user ID to look up', 'string') }}
```

### Dynamic Email Content
```javascript
// Email node body
Hello {{ $fromAI('userName', "The user's name") }},

Your order {{ $fromAI('orderId', "The order ID") }} has been processed.
```

### Dynamic Database Query
```javascript
// Database node query
SELECT * FROM users WHERE email = '{{ $fromAI('email', "The user's email to search for") }}'
```
